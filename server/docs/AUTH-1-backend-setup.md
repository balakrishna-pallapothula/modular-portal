# AUTH-1: Backend Foundation Setup for Authentication

**Related GitHub Issue:** [AUTH-1](https://github.com/balakrishna-pallapothula/modular-portal/issues/1)

---

## Overview

This issue establishes the **base backend infrastructure** to support authentication.  
It provides a clean, modular foundation using **Express** and **MongoDB**, without including any authentication logic yet.

---

## Scope

The following tasks are completed as part of this issue:

- Initialize Express server
- Setup environment configuration using `.env`
- Configure MongoDB connection
- Create modular folder structure
- Add health check endpoint (`/api/health`)

---

## Out of Scope

- Authentication logic
- User model
- JWT implementation
